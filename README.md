### Features :
* Displays a list of books
* View book details
* Added a new book
* Changing book data
* Delete books

## Prerequisites and Technology

* Node.js
* Hapi
* Nodemon
* Postman

## Install

```sh
npm install
```

## Hapi Installation and Nodemon

```sh
npm init --y
```

```sh
npm install @Hapi/hapi
```
```sh
npm install nodemon
```

## Usage

```sh
npm run start
```
